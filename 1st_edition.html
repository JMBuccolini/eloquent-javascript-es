Eloquent JavaScript

Una Introducción Moderna a la Programación

por Marijn Haverbeke

Obtén el libro

Eloquent JavaScript es un libro que proporciona una introducción al lenguaje de programación JavaScript y a la programación en general.


—Brendan Eich, el hombre que nos dio JavaScript

El libro existe en dos formas. Originalmente fue escrito y publicado en formato digital, que incluye ejemplos interactivos y un mecanismo para jugar con todo el código de ejemplo. Esta versión se publica bajo una licencia abierta.

He publicado una versión revisada del libro en papel. La estructura de esta versión se mantuvo en gran medida igual, pero todo el texto ha sido minuciosamente editado y actualizado. Puedes pedirlo en Amazon aquí. Todavía hay un entorno de codificación interactivo para esta versión, como una página separada. Las erratas están aquí.

La versión digital está disponible en los siguientes formatos:

  • HTML
  • HTML, comprimido
  • HTML, una sola página (no interactivo)

Las siguientes traducciones están disponibles:

  • Alemán (libro en tapa blanda)
  • Polaco (HTML)
